# ATS Connector for Resdex (Starter Files)

This folder contains the requested Chrome extension starter files and a copy of the Copilot prompt.

## Files

- manifest.json: Chrome extension configuration
- content.js: Profile scraper (Resdex preview page)
- background.js: Sync engine (captures profile + download hook)
- STEP_2_COPILOT_INSTRUCTIONS.txt: Exact message to paste into Copilot

Note: This is a starter scaffold. You will likely need to implement syncFileToFlask in background.js for file upload.
