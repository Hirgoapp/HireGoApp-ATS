## 🎓 Key Concepts (Quick Reference)

### Multi-Tenancy
- Single database, many companies
- company_id in every table
- Application-level isolation
- No cross-tenant data possible

### Custom Fields
- Metadata-driven customization
- JSONB storage
- No schema changes per customer
- Type-agnostic field definitions

### Soft Deletes
- deleted_at timestamp pattern
- Preserve audit trail
- Enable data recovery
- GDPR compliance

### Feature Flags
- License-based feature control
- Basic/Premium/Enterprise tiers
- Enable/disable per company
- Checked at route guard

### Modules
- 16 core modules
- Service layer pattern
- Dependency injection
- Inter-module communication
