## 💡 KEY FEATURES EXPLAINED

### Dashboard
Shows real-time metrics (company count, user count) fetched from backend APIs. Not dummy data - everything is live.

### Companies Management
Full CRUD with form validation, pagination, search, and real data binding to backend APIs.

### Users Management
Create, list, and delete super admin users with form validation and error handling.

### Settings
Configure platform settings like maintenance mode, session timeout, API rate limits.

### Authentication
JWT-based with automatic token refresh on 401. Auto-logout if refresh fails.

### API Client
Centralized axios configuration with request/response interceptors for token management.

### Form Validation
Zod schemas for runtime type safety and validation on all forms.

### Error Handling
Global error messages throughout the application with user-friendly explanations.
